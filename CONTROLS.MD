# Marker Pointing:

* F2 - FORWARDS
* F3 - BACKWARDS
*  ` - LEFT
* ALT - RIGHT
* UP ARROW - UP
* DOWN ARROW - DOWN
* ARROW LEFT - ROTATE CLOCKWISE
* ARROW RIGHT - ROTATE COUNTER-CLOCKWISE
* E - Save and Copy to Clipboard
